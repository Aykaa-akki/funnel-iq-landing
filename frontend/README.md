# Funnel-IQ Landing Page

## Deploy to Vercel

1. Login to Vercel: https://vercel.com
2. Import this folder
3. Framework: Create React App
4. Build Command: `yarn build`
5. Output Directory: `build`
6. Deploy!

## Environment Variables (Add later for Razorpay)

- `REACT_APP_BACKEND_URL` - Your backend API URL
- `REACT_APP_RAZORPAY_KEY` - Razorpay Key ID

## After Deployment

Connect your domain aykaa.me in Vercel settings.

Site will be available at: https://aykaa.me/funnel-iq/
